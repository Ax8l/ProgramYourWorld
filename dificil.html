Dificultad Difícil

Los lenguajes difíciles suelen ser usados por programadores avanzados que necesitan control profundo sobre el hardware y optimización. Ejemplos clásicos son C++ y Rust.

Mira este ejemplo básico en C++ que imprime un mensaje en pantalla:

#include <iostream>

int main() {
  std::cout << "Hola, Mundo!" << std::endl;
  return 0;
}

Este código incluye una librería estándar, define la función principal main, y usa cout para mostrar texto.

Lenguajes como C++ requieren entender conceptos más complejos como gestión de memoria, punteros y compilación.
