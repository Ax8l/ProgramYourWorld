¿Qué es un lenguaje de programación?

Un lenguaje de programación es como un idioma que usan las computadoras para entender instrucciones. Gracias a él, los programadores pueden decirle a la máquina qué hacer, desde mostrar un mensaje en pantalla hasta crear videojuegos o aplicaciones. Cada lenguaje tiene sus propias palabras y reglas, pero todos sirven para que las computadoras hagan tareas específicas. Algunos ejemplos famosos son Python, Java y JavaScript. En pocas palabras, un lenguaje de programación es la herramienta que transforma nuestras ideas en acciones que una computadora puede realizar.
// Función para saludar
function saludar(nombre) {
  return `Hola, ${nombre}! Bienvenido.`;
}

const mensaje = saludar('Mundo');
console.log(mensaje);
