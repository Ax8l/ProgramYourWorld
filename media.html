Dificultad Media

Los lenguajes de dificultad media son perfectos para quienes ya conocen lo básico y quieren crear aplicaciones más dinámicas y versátiles. Lenguajes como JavaScript y Java entran en esta categoría, siendo fundamentales para desarrollo web y aplicaciones.

A continuación, un ejemplo sencillo de una función en JavaScript que saluda:

function saludar(nombre) {
  return `Hola, ${nombre}!`;
}

console.log(saludar("Mundo"));

Este código define una función que recibe un nombre y retorna un saludo usando plantillas literales. Luego muestra el mensaje en la consola del navegador.

JavaScript es muy usado para hacer páginas web interactivas y mejorar la experiencia del usuario con animaciones, validaciones y más.
