Dificultad Fácil

Los lenguajes de programación fáciles son ideales para quienes empiezan en el mundo de la programación. Estos lenguajes tienen una sintaxis clara y sencilla, permitiendo concentrarse en aprender conceptos básicos sin demasiada complejidad.

Uno de los lenguajes más populares para principiantes es Python, gracias a su sintaxis limpia y legible.

Veamos un ejemplo básico de Python que muestra cómo crear una función que saluda a una persona:

def saludar(nombre):
    return f"Hola, {nombre}!"

print(saludar("Mundo"))

Este código define una función llamada saludar que recibe un nombre y devuelve un saludo personalizado. Luego, imprime el saludo para el nombre "Mundo".

Este tipo de lenguaje es perfecto para entender cómo funcionan las funciones, variables y la estructura básica del código.
